# Sysh (Syscall shell)

This project was made entirely for the purpose of learning x64 assembly.

To assemble, you must be on an x64 linux system, or in WSL and have `nasm` installed on your computer.
If you do not have `nasm`, simply run `sudo apt install nasm`.
Once nasm in installed, run `build.sh` and the shell will appear as `./build/main`.

### Currently Supported Commands

Please note that flags are not currently supported.

- ls
- cd
- pwd
- cat
- (execution)
